    # 关于离线模式

    离线模式已完全从启动器中移除.你仍然可以创建离线账户但你将不能进行游戏.你需要 Microsoft/Mojang 账号来购买Minecraft Java版.

    Pojavlauncher不会支持任何形式的盗版活动

    # 前往 Minecraft.net (opens new window)购买正版Minecraft.

    如果你想免费游玩Minecraft,可在此下载 (opens new window)或使用中国版 (opens new window)
