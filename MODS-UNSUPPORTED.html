# 不支持的模组

PojavLauncher并不完全支持所有Minecraft模组,以下提供了不完全的不支持模组列表:

  • Sodium (opens new window) (不支持GL4ES渲染器, 支持ANGLE与virglrenderer渲染器)
  • Rubidium (opens new window) (与Sodium一样^)
  • Chlorine (opens new window)
  • Create (opens new window)
  • Scannable (opens new window)
  • Colormatic (opens new window)
  • ItemPhysic (Lite (opens new window) works)
  • RandomPatches (Forge) (opens new window) (Fabric) (opens new window)
  • Immersive Vehicles (opens new window) (Forge Only)
  • MrCrayfish's Gun Mod (1.16.5) (opens new window)
  • ReplayMod (opens new window) (FFMPEG wrongly placed)
  • Dynamic Surroundings (opens new window) (white textures)
  • Sex mod (thanks @Adiixyz)
  • Essential (opens new window)
  • MysteryMod (opens new window) (Not fully tested)
  • Skytils (opens new window) (1.3.0-pre5 works)
  • Patcher (opens new window)
  • NotEnoughUpdates (opens new window)
  • BatMod (opens new window)
  • LabyMod (opens new window) (only 1.8.9)
  • Meteor Client (opens new window)

大多数不支持模组列表从 PojavLauncher#1948 (opens new window) 获取
